## Project Purpose

This is a personal health and wellness documentation repository in Czech, maintained by Martin Milan. It documents his medical/psychological situation, therapeutic strategies, and communication needs — intended for sharing with AI assistants and healthcare providers.

## Working with Martin

- **Written requests preferred:** Short bullet points or brief sentences. Avoid long paragraphs.
- **Frequent breaks:** Martin needs reminders to take small breaks, hum something, or breathe — especially after each exchange. Reference topic `0112` for digestive/peristalsis health.

## File Naming Conventions

Files follow a numeric prefix + descriptive name + suffix system:

- `01xx-` — Core support themes (`011` = help/support, `0112` = digestive health subtopic, `0119` = rest/break needs)
- `0b-` — Hospital avoidance topic
- `02-` — Secondary wellness topics
- `.publ.md` — Published/public version of a document

- aaa nn - `.f8.md`, `.f5`, `.f4` — File type/version codes

- `PUBL` in filename — marks files intended for external sharing

YAML front matter in files often contains: song titles (healing music), emoji state markers (💚⏰🎵), and tags like `resting`, `recovery`, `restore`, `music`.

## Repository Structure
